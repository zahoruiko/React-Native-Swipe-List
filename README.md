# React-Native-Swipe-List
# An example of implementing a list with the function of swiping its elements to the right and left in React Native

# Getting Started

## Install modules:

```npm install```

### Install for iOS:

```
cd ios
pod install
```

## Start

```bash
# using npm
npm start

# OR using Yarn
yarn start
```
