# React-Native-Swipe-List
# An example of implementation of a list with a function for swiping its elements in React Native

# Getting Started

## Install modules:

```npm install```

### Install for iOS:

```
cd ios
pod install
```

## Start

```bash
# using npm
npm start

# OR using Yarn
yarn start
```
